# Vanta Visuals

Сайт для Vanta Visuals — премиальные визуальные эффекты.

## Структура

- `index.html` — Главная страница с приветствием и FAQ
- `facts.html` — Страница с фактами о продукте
- `buy.html` — Страница с тарифами (1 месяц, 6 месяцев, навсегда)
- `login.html` — Страница авторизации
- `profile.html` — Профиль пользователя
- `css/style.css` — Стили (чёрно-фиолетовая тема)
- `js/main.js` — Логика авторизации, профиля, FAQ

## Запуск

Откройте `index.html` в браузере или используйте любой локальный сервер:

```bash
npx serve .
```

## Деплой через Railway

1. Создать репозиторий на GitHub и запушить код
2. На Railway создать проект → PostgreSQL (база данных)
3. Создать сервис для бэкенда (Node.js + Express)
4. Подключить переменные окружения DATABASE_URL из PostgreSQL
5. Настроить deploy из репозитория
